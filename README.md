## Getting Started

1. Install dependencies

   ```
   npm install
   ```

2. Run the development server

   ```
   npm run dev
   ```

3. Open in your browser:  
   [http://localhost:3000](http://localhost:3000)

---

## Project Structure

| Folder | Description |
|---------|--------------|
| `app/` | Main application pages and layout |
| `public/` | Static assets (SVGs, images) |
| `ui/` | UI components (NavBar, Footer, ProductCard) |
| `utils/` | Utility constants |
| `types/` | Type definitions |

---

## Customization

- Add products in the PRODUCTS array (`page.tsx`)  
- Update testimonials in the TESTIMONIALS array  
- Modify styles in globals.css and Tailwind config
